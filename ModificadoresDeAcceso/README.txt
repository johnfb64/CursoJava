Programa de prueba que demuestra:

- Se genera atributo en clase 1 y se instancia en clase 2, lo que permite hacerlo sin problemas al estar en el mismo paquete. 

- De la clase 3 se debe importar paquete. 

- Para que el atributo sea reconocido en la clase 3, se debe importar la clase del paquete y se debe hacer publico el atributo. 
import Pack1.Class1;

- Los modificadores de acceso pueden ser public o private.
